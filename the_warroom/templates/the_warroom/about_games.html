{% trans 'About Games' %}

{% trans 'Games' %}

{% blocktrans %} A Game is a record of a played game of Root. It captures the key details of each session including which Map and Deck were used, the Platform it was played on (Tabletop Simulator, Root Digital, or In Person), and any optional components like Hirelings, Landmarks, or House Rules. {% endblocktrans %}

{% blocktrans %} Games can be recorded as standalone or as part of a Series. When part of a Series, game results contribute to leaderboards and bracket advancement. Any logged-in user can record a game. {% endblocktrans %}

{% trans 'Seats' %}

{% blocktrans %} Each player or faction in a game is represented by an Seat. A Seat records which Player played which Faction, their final score, and whether they won the game. Seats are recorded in turn order. {% endblocktrans %}

{% blocktrans %} Seats also track special win conditions like Dominance victories and Coalitions. {% endblocktrans %}

{% blocktrans %} For Vagabonds, the Vagabond's character is recorded as well. {% endblocktrans %}

{% trans 'Scorecards' %}

{% blocktrans %} A Scorecard provides a detailed turn-by-turn breakdown of how a faction scored their points in a game. Each turn can track points across several categories: {% endblocktrans %}

  • {% trans 'Battle points: points gained from removing cardboard (buildings or tokens)' %}
  • {% trans 'Crafting points: points gained from crafting items' %}
  • {% trans 'Faction points: points gained from faction-specific abilities' %}
  • {% trans 'Other points: points from other sources (e.g. The Tower or Charm Offensive)' %}

{% blocktrans %} Or if you prefer you can choose to use the simplified view to record points without worrying about the category. {% endblocktrans %}

{% blocktrans %} Scorecards can be linked to a Seat in a game, connecting the turn-by-turn data to the overall game record. If they are not linked they will show up in the "In Progress" tab of the Battlefield. {% endblocktrans %}

{% blocktrans %} Scorecards can be recorded during or after a game is recorded. To set up Scorecards while you play, click "Record Socrecard" in the Battlefield. Click "Save and Add New Seat" to add the next faction. Once all factions have been added you can quickly rotate through the seats using the navigation buttons at the bottom of the page. {% endblocktrans %}
